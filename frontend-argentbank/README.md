# Frontend -Argent Bank

Ce dépôt contient la partie frontend de l'application ArgentBank pour le projet 10 du parcours OpenClassrooms "Développeur d'application React"

## Description

Frontend dévélopé en React/Vite pour une application bancaire. 
L'interface permet de se connecter, consulter le profil et afficher les transactions.

## Installation

1. Installer les dépendances :
   ```bash
   npm install
   ```

2. Démarrer le serveur de développement :
   ```bash
   npm start
   ```

## Structure

- `src/` : code source React
- `public/` : fichiers statiques

## Usage

Accéder à l'application dans le navigateur à `http://localhost:5173/`

## Notes

Ce frontend dépend de l'API backend pour l'authentification et les données des comptes, fournies par le projet 10 du parcours
